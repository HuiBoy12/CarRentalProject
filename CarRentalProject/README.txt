This is a simple project to manage car rentals according to the deliverables and requirements from Al. This project includes:

1. SQL Folder contains SQL necessary to create the database and tables in MySQL server and insert data into the database. Additionally, it includes 
the EER digram used to design the database
2. Templates contains the html files for the website where a user can dsiplay, add, update, and remove a customer's information
3. The python files that implement the system 

To launch the website, run front_end.py and click on the link generated by the terminal
